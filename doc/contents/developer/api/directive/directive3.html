탭 Tab

탭 컴포넌트는 bootstrap3 css를 이용하여 탭 기능을 제공하는 지시자입니다.

Usage

텝 컴포넌트는 기본적으로 아래와 같이 태그를 입력하여 적용합니다.

parameter

텝 컴포넌트의 주요 파라메터는 아래와 같습니다.

텝 설정 정보(상위 div)
egov-tab
텝 컴포넌트를 명시합니다.
direction
텝 페널이 렌더링 되는 방향을 설정합니다. "right","left","below"중 하나를 설정해야 합니다.
텝 페널 설정(하위 div)
heading
텝 패널의 타이틀 문자열이다. 별도의 egov-panel 지시자를 사용하지 않으면 필수적으로 입력해야 한다.
active
optional
페널의 활성화 여부
disabled
optional
페널의 비활성화 여부
select
optional
페널을 선택할 경우 호출되는 함수명
